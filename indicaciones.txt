Build de TypeScript

1. Instalar Node.js
2. Instalar TypeScript
3. Crear un proyecto nuevo con `tsc --init` y configurar el archivo `
4. Crear un archivo `index.ts` con el código de la aplicación
5. Compilar el código con `tsc index.ts` y ejecutarlo con `node
